## Project Description

SamTech Solutions is a complete, responsive, multi-page website for a fictional small IT consultancy. It helps small businesses and startups explore professional website development, software solutions, IT consulting, and data services before requesting a free consultation.

The project demonstrates semantic HTML5, external CSS3, responsive design, accessible forms, structured tables, and client-side JavaScript interaction.

## Pages

- Home (`index.html`)
- About (`about.html`)
- Services (`services.html`)
- Contact (`contact.html`)

## JavaScript Features

- Accessible mobile navigation
- Service selection carried from service links into the contact form
- Contact-form validation with clear field errors
- Live message character counter
- Honest in-page confirmation for the static demonstration
- Automatic copyright year

## Privacy Note

The consultation form is a browser-only demonstration. It validates and summarises input but does not transmit or store personal information.
