# SamTech Solutions Privacy Spec

## Data Boundary

This is a static student project hosted on GitHub Pages. It has no database, server, analytics system, authentication, or payment processing.

## Contact Form

- Fields are processed only in the visitor's browser for validation and confirmation.
- Form values are not stored, uploaded, emailed, or shared.
- The page must explain this limitation before submission.
- The confirmation must not falsely claim that a message was delivered.

## External Resources

- Prefer local assets and system fonts to avoid unnecessary third-party requests.
- Do not include tracking scripts, advertising pixels, or embedded third-party forms.

## Claims

- Sample prices are estimates for demonstration.
- Do not invent testimonials, client logos, certifications, addresses, or completed projects.
